Frontend – React (Typescript) + Tailwind
Desenvolva as seguintes telas obrigatórias:
✔ Tela de Login
• Acesso para administrador e funcionários
✔ Tela de Cadastro de Funcionário (somente para administrador)
• Nome
• E-mail
• Senha
• Cargo ou função (opcional)
• Definir tipo de usuário: funcionário
• Essa tela pode ser simples, mas funcional
✔ Dashboard com Kanban
Colunas:
• A Fazer
• Fazendo
• Concluídas
✔ Formulário de Criar e Editar Tarefas
• Disponível para administrador
• Funcionário pode editar tarefas atribuídas a ele (opcional)
✔ Dashboard de Métricas (se possível)
• Gráficos ou cards
• Resumo da produtividade do time
• Total de tarefas concluídas
• Dados por funcionário
